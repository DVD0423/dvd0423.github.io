Python实现实验室自动日报生成器

发表于 2016-04-25   |   分类于 python   |  

在我大宇宙实验室，每天必须打卡上班，而且每天必须登录实验室网站写每日工作记录，即QA。而我经常忘记写QA，所以想着用Python写一个自动化脚本挂在实验室另一台服务器上，每天12点post一次我的日常工作内容到QA服务器。

本程序使用requests模块和python3.5。使用chrome开发者工具分析网站行为，可以得出post的cookies生成方式。区区几十行代码就实现了该功能。不得不说Python真是一个生活工具，写爬虫、写自动化小工具，代码简介，上手迅速。
