Python爬取美拍视频

发表于 2016-04-28   |   分类于 python   |  

　　前段时间看到美拍上有很多hiphop街舞视频，所以想下载下来学习一下。于是就产生了用python批量下载的想法。花了一个下午分析了美拍网站的html信息和视频加载机制。成功实现了爬取带“hiphop”关键字的视频。

1 看美拍视频的操作流程

　　首先我们要明白：爬虫的目的就是用机器模仿人工操作，实现web的自动化请求（Request）与响应（Response）。因此我们首先要做的是熟悉人工操作的执行流程。
　　第一步当然是先打开美拍首页，点击进入舞蹈相关页面。在视频多列模式下，美拍一页只能容纳24个视频，如果需要看到更多的视频需要把页面往下拉，这时会触发ajax异步刷新机制，实现页面的局部刷新，从而get到更多的视频。当页面连续刷新8次时，就达到了页面底部，这时不在刷新更多的视频，也就是说一个页面所列出的视频是有限的，最多为8 24个。但是页面底部会出现第一页、第二页的选项。所以，*如果要继续看到更多的视频需要点击下一页按钮。
　　只要重复上面的操作就可以欣赏到美拍上的所有舞蹈视频。这一章节都是很简单的操作，却也是一个爬虫成形最关键的地方。
