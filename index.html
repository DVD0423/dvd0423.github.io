1 变量命名规则

双引号内的特殊字符，如$，会被翻译；而单引号内的所有字符保持不变。

使用命令给变量赋值方式：$(命令)或者`命令`

为变量增加变量内容可以使用:

变量默认是字符串，整数的定义declare -i var；常量的定义：declare -r var

2 输入输出

变量输入：read var
提示输入：read -p "input" filename

echo -e 解释转义字符。

echo -n等价于echo -e "\c"不换行输出。

print格式化输出: printf '%s\t %s\t\n' $(cat printf.txt)

3 ulimit限制资源使用

4 bash通配符

符号 意义
* 0到无穷多个任意字符
? 代表一定有任意一个字符
[abc] abc内的任意一个
[^abc] 非abc内的任意一个
[a-z] a到z

5 数据流重定向

  • /dev/null 垃圾桶，可以把标准输出和错误输出都放到这里
  • 1> 或者 > 覆盖标准输出到文件
  • 1>> 或者 >> 累加标准输出到文件
  • 2> 或者> 覆盖错误输出到文件
  • 2>> 或者>> 累加错误输出到文件
  • cat > ./myfile < diff.txt 把diff.txt输出到myfile文件
  • cat > ./myfile << "eof" 遇到eof则键盘输入结束
  • &> /dev/null 或者 > list 2>&1
  • 1> /dev/null 2> /dev/null
  • ls -l | tee myfile 输出结果存一份到myfile
  • ls -l | tee -a myfile 输出结果累加存一份到myfile

6 grep

参数或正则 意义
-v 反向查找
-n 显示行号
^word word在行首
word$ word在行尾
‘\.\$’ 显示以”.”为结尾的行
. 一定有一个任意字符
* 重复0到无穷多个前一字符
.* 重复1到无穷多个前一字符
\{n,m\} 连续n到m个的前一个RE字符
\{n,\} 连续n个以上的前一个RE字符
\{n\} 连续n个前一个RE字符

7 sed

参数或正则 意义
-n 只有经过特殊处理的内容才会被显示
-i 直接修改读取的文件内容，而不是由屏幕输出;sed -i '$a # this is a test' myfile.txt在文件最后添加字符串# this is a test
ls | sed ‘2,5d’ 删除2-5行
ls | sed ‘2a fuck DW’ 在第二行加入字符串fuck DW
ls | sed ‘2,5c fuckDW’ 把2-5行替换成fuckDW
ls | sed -n ‘2,7p’ 显示2-7行，一般p和-n参数一起使用
sed ‘s/旧字符串/新字符串/g’ 字符串替换

正则表达式使用方法和grep一样

8 awk

变量名 意义
\$0 一整行数据的意思
\$1 \$2 表示每行的第几段
NF 每一行（$0）拥有的字段总数
NR 目前awk处理的是“第几行”数据
FS 目前的分割字符，默认是空格键

首先我们要明白：爬虫的目的就是用机器模仿人工操作，实现web的自动化请求（Request）与响应（Response）。因此我们首先要做的是熟悉人工操作的执行流程。
